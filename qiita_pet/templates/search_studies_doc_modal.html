A basic search consists of three parts: The metadata category to search over, the operator to use, and the value to use in the search. For example, to get all soil samples in the database, the query would be:

sample_type = ENVO:soil

Valid operators for searches are:

< = <= >= includes

The mathematical operators work as expected. Includes is used to do substring searches.

Complex queries can also be created by using AND, OR, and NOT operators. As an example, if we want all soil samples that are extremely low or extremely high pH, we can use the following search:

sample_type = ENVO:soil AND (ph < 4 OR ph > 10)
